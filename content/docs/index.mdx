## How to use this site

- Lessons are ordered. If something later confuses you, the answer is usually in an earlier section.
- Every command is meant to be **typed**, not copy-pasted, on your Fedora box. Muscle memory matters.
- Callouts like the one below are the most-skipped, most-important content.

> **Tip** — Keep a plain text file open while you work. Write down every command you don't yet understand. Look them up at the end of each session. By section 3 you'll be reading them like English.

## What you'll have when you finish

- A hardened, internet-reachable Linux server you fully understand
- A real Node.js or Python app you deployed yourself, behind HTTPS, on your own domain
- A monitoring + backup setup that actually works
- The confidence to rent a cloud VPS, harden it in an hour, and run real workloads on it
