# Networking

Networking is where most beginners get stuck because the vocabulary feels overwhelming. We'll fix that with three concrete lessons.

1. [IPs, Ports & DNS](/networking/ip-ports-dns) — The address system the entire internet runs on, in plain English.
2. [Firewall (firewalld)](/networking/firewalld) — Decide exactly which ports the world can knock on.
3. [Hardening SSH](/networking/ssh-hardening) — Stop the password brute-forcers cold.

> **The one sentence that explains networking** — Every server is reachable at an **IP address**, on a numbered **port**, and DNS is a phonebook that turns names into IPs. Almost every networking problem reduces to "which of those three is wrong?"
