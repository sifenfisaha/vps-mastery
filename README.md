# VPS Mastery

A complete, hands-on path from beginner to advanced Linux server administration — written for developers who want to understand servers instead of just renting them.

The course uses a spare PC running Fedora as a personal lab, so you can practice every concept on a real machine without renting cloud infrastructure. Every skill transfers directly to a production VPS at Hetzner, DigitalOcean, AWS, or anywhere else.

## What you will learn

- **Foundations** — what a VPS really is, how a Linux server is structured, and your first SSH connection.
- **Linux Basics** — the shell, file permissions, users and `sudo`, package management, processes, and `systemd`.
- **Networking** — IP addresses, ports, DNS, firewalls, and hardening SSH against the open internet.
- **Remote Access** — reaching your server from anywhere with Tailscale, Cloudflare Tunnel, or port forwarding.
- **Hosting Applications** — Nginx and Caddy, TLS certificates, deploying a Node.js app, PostgreSQL, and Docker.
- **Operations** — reading logs with `journalctl`, monitoring, backups, and scheduling jobs with `cron` and `systemd` timers.
- **Advanced** — Docker Compose, multi-app reverse proxies, CI/CD to a server, a full hardening checklist, and graduating to a real cloud VPS.

## Who this is for

Developers, students, and self-hosters who:

- Are comfortable using a computer but new to running their own servers.
- Want to understand the building blocks behind the apps they deploy.
- Prefer learning by doing, on real hardware, with explanations that don't skip the "why."

No prior Linux experience required. By the end you will be able to deploy, secure, monitor, and back up real services on a Linux server with confidence.
